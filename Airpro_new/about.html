About AirPro

AirPro Smart Helmet is an IoT-based safety and air quality monitoring system. It continuously monitors AQI, battery level, accident detection, and fan system status using real-time Firebase database integration.

Designed for riders and industrial workers, AirPro enhances safety through smart alerts, live monitoring dashboards, and intelligent environmental analysis.

Vision

Create intelligent wearable systems for real-time safety monitoring.

Technology

Built using Firebase, IoT sensors, and modern web technologies.
